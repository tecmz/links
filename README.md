# 魔众资源导航系统

魔众资源导航系统提供一个可以快速构建在线资源导航网站的源码系统，支持私有化部署。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
